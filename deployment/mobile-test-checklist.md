# 手机端上线测试清单

## 基础加载

- [ ] 首次打开域名能进入游戏页面。
- [ ] 强制刷新后仍能打开。
- [ ] 第二天再次打开仍能进入。
- [ ] 首页 3 秒内出现游戏画面。
- [ ] 弱网下不会长时间白屏。

## 网络覆盖

- [ ] 中国移动 4G/5G。
- [ ] 中国联通 4G/5G。
- [ ] 中国电信 4G/5G。
- [ ] 家庭 Wi-Fi。
- [ ] 校园/公司 Wi-Fi。

## 游戏功能

- [ ] 开始界面显示“开始游戏 / 开启音乐 / 声音：开 / 清空榜单”。
- [ ] 点击“开启音乐”后开始前音乐播放。
- [ ] 点击“开始游戏”后游戏中音乐播放。
- [ ] 左右、加速、刹车按钮可用。
- [ ] 暂停、继续、重开可用。
- [ ] 游戏结束后可提交本地成绩。
- [ ] 清空榜单可用。

## 离线与缓存

- [ ] 首次成功打开后，关闭网络再打开能显示游戏壳。
- [ ] 更新版本后，带 `?v=提交号` 的链接能看到新版。
- [ ] `sw.js` 可以直接访问。

## 发布检查

- [ ] CDN HTTPS 证书有效。
- [ ] `index.html` 返回 200。
- [ ] `sw.js` 返回 200。
- [ ] `assets/audio/start.mp3` 返回 200。
- [ ] `assets/audio/race.mp3` 返回 200。
- [ ] 浏览器控制台没有 404。
